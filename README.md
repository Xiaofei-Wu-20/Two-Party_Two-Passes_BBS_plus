# Two-Party_BBS_Plus_Signature_in_Two_Passes_Implementation
This repository provides the official C++ implementation of the scheme described in our paper:

"Two-Party BBS+ Signature in Two Passes", accepted by ESORICS 2026.

## Overview
This project focuses on the performance and efficiency of our two-party BBS+ signature scheme that requires only two communication passes. 
The core protocol logic, security primitives, and performance benchmarking tests are all contained within the src directory.
